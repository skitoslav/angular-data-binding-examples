Description

Based in component strategy provides the fastest way of building a new functionality.
There is no need in any other components, services or more complex architect solutions.

This solution is worth it when you need to create the MVP of some functionality.
Or when you create something tiny and without any future plans for expansion.

Code example


        export class YourComponent {
          users$: Observable<IUser[]>

          constructor(private readonly httpClient: HttpClient) {
            this.users$ = this.httpClient.get<IUser[]>('api/users');
          }
        }
